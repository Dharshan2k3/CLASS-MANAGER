Class Manager

A Student-Teacher Portal

About Us

Class Manager is a student-teacher portal with various classroom features like uploading assignments, submissions, Class Notice and many more. Students and Teacher can login in the portal and can experience interactive classroom experience online. One of the most powerful features of class manager is its student managing abilities. Teacher can easily manage their students and keep them updated about all of the activities in class.

Our Services

For Teacher

  1. Teachers can add students to their class.
  2. Teachers can give or edit student marks.
  3. Teachers can write notices sent to all students.
  4. Teachers can upload assignments.
  5. Teachers can check student assignment submissions and grade them.
  6. Students can message teachers directly.
  7. Teachers can view and edit student marks.
  8. Users can add a profile picture through the edit profile option.

For Student

  1. Students can log in and stay updated on class activities.
  2. Students can view notices from their teachers.
  3. Students can download and submit assignments.
  4. Students can contact teachers through messaging.
  5. Students can check their grades.
  6. Users can add a profile picture through the edit profile option.

Digital Learning

We surveyed over 16,000 teachers and administrators to better understand the state of digital learning. Class Manager provides all facilities that are needed for improving digital learning.
